# gui 简单的scssUi库
  * 一套通用的 scss-mixin 库，方便快速进行浏览器适配
  * 一套通用的函数式编程库  g-lib
  * 多种在angular中使用第三方库的方式
  * Rx 响应式编程常用操作
  
## 目录结构

```js
  --app
    --glib                    // 函数库
    --component               // 组件
    --core                    // 核心模块
    --shared                  // 共享模块
    --page                    // 路由页面
    --module                  // 懒加载模块
    --three                   // 第三方js库
  --assets                       // 静态资源目录
    --fonts                       // 字体图标
    --mixin                       // scss库
    --svg                          // svg文件(不建议直接使用,可以先转换成字体图标)
    --jquery
    --bootstrap
```
