# Typescript Tool library

    A library of data manipulation and processing for JS platform
    
### Use

    npm i math-ts --save

```typescript

    import * as math from 'math-ts'; 
    const arr = [1, 2, 3, 4];
    math.chunk(arr, 2);
        
```
